
## Installation :

* `apt update`
* `apt install git curl -y`
* `git clone git://github.com/Yahyaamanj/track-ip.git`
* `cd track-ip`

#### > Run : `bash trackip`

## Single Command :
```
apt update ; apt install git curl -y ; git clone git://github.com/Yahyaamanj/track-ip.git ; cd track-ip ; bash trackip
```
